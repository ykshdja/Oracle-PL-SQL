# Oracle PL/SQL Practice Repository

This repository contains structured **Oracle PL/SQL programs** written to
practice and demonstrate core database programming concepts used in
enterprise systems.

## Overview
The goal of this repository is to build a strong foundation in **procedural SQL**
by implementing common PL/SQL constructs such as procedures, functions,
triggers, cursors, and exception handling.

The code focuses on **clarity, correctness, and real-world usage patterns**
rather than short competitive-style scripts.

## Topics Covered
- Anonymous PL/SQL blocks
- Stored procedures
- User-defined functions
- Cursors (implicit and explicit)
- Triggers
- Exception handling
- Control structures (loops, conditionals)
- Basic SQL integration with PL/SQL

## Tech Stack
- Database: Oracle Database
- Language: PL/SQL
- Tools: SQL*Plus / Oracle SQL Developer

## Repository Structure
Each file demonstrates a specific PL/SQL concept or use case.
Programs are written independently to keep examples focused and readable.
